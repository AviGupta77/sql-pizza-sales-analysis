# Pizza Sales Analysis using SQL

## Project Overview
This project analyzes pizza sales data using SQL to extract meaningful business insights such as revenue trends, popular pizzas, and category-wise performance.

## Dataset
This project uses multiple relational CSV files including orders, order details, pizzas, and pizza types for SQL-based analysis.


## Business Questions Answered
- Total revenue generated
- Most popular pizza types and sizes
- Category-wise sales distribution
- Daily and monthly order trends
- Cumulative revenue over time

## SQL Concepts Used
- SELECT, WHERE
- JOINs
- GROUP BY, HAVING
- Aggregate Functions
- Subqueries
- ORDER BY

## Key Insights
- Certain pizza categories contribute significantly to total revenue
- Medium and large pizzas are the most frequently ordered
- Sales show clear trends across different time periods

## Tools Used
- MySQL
- SQL
